# habari-habour
Habari Habour is a Social Media curation website offering content creators the trending news

## Structure

|-- .virtual_environment
|-- app # the main module to store the app static pages, templates, static css, 
|  |-- __init__.py 
|  |-- auth.py # the routes for the app
|  |-- db.py # the database creation function
|  |-- schema.sql # the sql structure
|  |-- interest.py 
|  |--- templates # the template html files
|  |   |-- auth # the authentication html
|  |   |   |-- login.html
|  |   |   |-- register.html
|  |   |-- base.html
|-- tests # the test files for the modules
|-- .gitignore # the ignored files
|-- requirements.txt # the python libraries used to make Habari Habour
